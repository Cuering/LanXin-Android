# 真机回归清单（增量）

> 用途：合并后 / 发版前，按条目在真机快速冒烟。  
> 本文件只记**可手测**的产品路径；CI 已覆盖的单测不重复。

## 知识库 · 文件夹批量导入（#69）

入口：**设置 → 知识库**

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 点「选择文件」，选单个 `.txt` / `.md` / `.pdf` | 进度可读；完成后 snackbar 含段数；向量条数增加 |
| 2 | 点「选择文件夹」，选含嵌套子目录的目录（内有 `.md` / `.txt`） | 扫描后逐文件导入；UI 显示 `文件 done/total · 成功 n` |
| 3 | 选空文件夹或无可支持格式的文件夹 | 失败提示：无可导入的 `.txt` / `.md` / `.pdf` |
| 4 | 导入进行中再点「选择文件/文件夹」 | snackbar：正在导入中，请稍候（互斥） |
| 5 | 取消系统文件夹选择器 | snackbar：未选择文件夹；进度保持 IDLE |
| 6 | HelpCard 文案 | 同时说明「选择文件」与「选择文件夹」及递归扫描格式 |
| 7 | 同一文件再次导入 | 覆盖对应条目（不无限膨胀同名 source） |
| 8 | 完成后点「清除进度」/「刷新状态」 | 进度复位；状态数字刷新正常 |

### 边界（可选）

- 超深目录树：深度上限约 6，过深子树可被跳过且不崩
- 混合格式文件夹：仅导入支持扩展名，其它静默跳过
- 部分文件解析失败：总结果仍可 DONE，显示成功/失败文件数
